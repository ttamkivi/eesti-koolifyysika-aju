# Eesti Füüsika Aju 🇪🇪

Eesti üldhariduskoolide füüsika **tsentraliseeritud teadmistebaas** — üks koht, kus hoitakse õppekava, teooriaid, ülesandeid, praktiliste tööde juhendeid ja õpetajate arhiivi.

---

## Mis see on?

See repositoorium on **elav dokument** — mitte lõpetatud õpik, vaid kasvav kogu, kuhu iga õpetaja saab panustada. Selle eesmärk on:

- Olla **selgroog** AI-põhise füüsikatuutori jaoks
- Olla **jagamisplatvorm** Eesti parimatele füüsikaõpetajatele
- Olla **abivahend** õpilastele iseseisval õppimisel

---

## Struktuur

```
/oppekava      → Ametlikud kursuse kaardid koos valemite ja õpitulemustega
/teemad        → Sisuline materjal: teooria, ülesanded, väärarusaamad
/eksamid       → Varasemad eksamid ja näidisvastused
/opetajate-arhiiv → Õpetajate isiklikud materjalid
/agendid       → AI-agentide juhendid (tulevikus)
```
