# Eesti Füüsikaolümpiaad — Juhend

## Olümpiaadi struktuur

### Võistlusvoorud

| Voor | Millal | Kus | Formaat |
|------|--------|-----|---------|
| **Koolivoor** | november–jaanuar | igas koolis | Kooli komisjon hindab |
| **Piirkondlik voor** | veebruar | TalTech / piirkonnakeskused | ~10 teooriaülesannet + 2 eksperimentaalülesannet |
| **Lõppvoor** | aprill | Tartu, Physicum | 2 päeva: teooria + eksperiment |
| **Lahtine võistlus** | november–detsember | TÜ teaduskool | 10 ülesannet, 6 parimat arvestatakse |

### Vanusegrupid

- **Põhikool** — 8.–10. klass
- **Gümnaasium** — 11.–12. klass

### Lõppvooru pääsemine

1. Eelmise aasta lõppvooru võitjad (8.–11. kl)
2. Piirkondliku vooru mõlema vanusegrupi parimad
3. TÜ teaduskooli lahtise võistluse parimad
4. Ülejäänud kohad riikliku pingerea alusel

## Ülesannete tüübid

### Teoreetilised ülesanded
- Nõuavad füüsikaseaduste rakendamist, matemaatilist modelleerimist ja loogilist arutlust
- Tüüpiliselt 10 ülesannet, millest 5 parimat arvestatakse
- Lahendusaeg: ~5 tundi (jagatud kahe päeva vahel lõppvoorus)

### Eksperimentaalülesanded
- Praktilised mõõtmised ja andmeanalüüs
- Tüüpiliselt 2 ülesannet, millest 1 parim arvestatakse
- Nõuavad: mõõtmiste tegemist, andmetabelite koostamist, graafikute joonistamist, veaanalüüsi

### Hindamisülesanded (estimation problems)
- Suurusjärgu hinnangud (Fermi ülesanded)
- Näide: "Kui palju vett voolab Emajõest läbi päevas?"

## Hindamiskriteeriumid

### Üldpõhimõtted

| Kriteerium | Kirjeldus |
|------------|-----------|
| **Füüsikaline mudel** | Kas ülesande füüsika on õigesti tuvastatud ja mudel valitud? |
| **Valemid ja seosed** | Kas vajalikud valemid on kirja pandud ja korrektselt rakendatud? |
| **Matemaatiline lahendus** | Kas arvutused on korrektsed ja loogilised? |
| **Vastus ja ühikud** | Kas lõppvastus on numbriliselt õige ja ühikutega? |
| **Eksperimentaalne töö** | (praktilistes ülesannetes) Mõõtmiste kvaliteet, andmete esitus, veaanalüüs |

### Osaliste punktide süsteem

- Õige füüsikaline lähenemine ilma lõppvastuseta → osaline punkt
- Arvutusviga õige meetodi juures → suurem osa punkte säilib
- Ainult lõppvastus ilma lahenduskäiguta → 0 punkti
- Hindamisskeem avaldatakse pärast vooru koos lahendustega

### Hindamise protsess (lõppvoor)

1. **Mentorite hindamine** — mentorid annavad esialgse punktisumma
2. **Kaitsmine komisjoni ees** — õpilane kaitseb oma lahendust erapooletu komisjoni ees
3. **Lõplik punktisumma** — komisjon kinnitab

## Teemade jaotus voorude kaupa

### Põhikool (8.–10. klass)

| Teema | Osakaal |
|-------|---------|
| Mehaanika (kinemaatika, dünaamika, jäävusseadused) | ~40% |
| Soojusõpetus | ~15% |
| Elektriõpetus | ~20% |
| Optika | ~15% |
| Hindamisülesanded | ~10% |

### Gümnaasium (11.–12. klass)

| Teema | Osakaal |
|-------|---------|
| Mehaanika (sh pöörlemisdünaamika, fiktiivjõud) | ~30% |
| Elektromagnetism | ~20% |
| Termodünaamika | ~15% |
| Optika ja lained | ~15% |
| Kaasaegne füüsika (kvant, tuuma) | ~10% |
| Hindamisülesanded | ~10% |

```
                              Eesti-Soome võistlus  →  IPhO
```
