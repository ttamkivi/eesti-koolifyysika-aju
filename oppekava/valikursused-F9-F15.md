# F9 — Jäävusseadused. Voolised

| | |
|---|---|
| **Kursuse kood** | F9 (Flai1) |
| **Eelduskursused** | F1–F3 |
| **Õpetamise aeg** | 10. klass |
| **Hindamisviis** | Mitteeristav (arvestustöö) |

## Eesmärgid

Kompleksülesanded staatikast ja jäävusseadustest. Jõumomendist tulenev muutuv ringliikumine. Voolistega seotud nähtused.

## Põhiseosed

$$\omega = \frac{\varphi}{t} \qquad v = \omega r \qquad a_n = \omega^2 r = \frac{v^2}{r}$$

$$\varepsilon = \frac{\omega - \omega_0}{t} \qquad \varphi = \varphi_0 + \omega_0 t + \frac{\varepsilon t^2}{2}$$

$$M = Fd \qquad \vec{p} = m\vec{v} \qquad E = E_k + E_p$$

$$p + \rho gh + \frac{\rho v^2}{2} = \text{const} \qquad L = J\omega \qquad J\omega = \text{const}$$

## Õppesisu

Staatika kompleksülesanded. Ühtlaselt muutuv ringliikumine. Impulsi jäävuse seadus. Impulsimoment. Bernoulli seadus. Ideaalne vedelik.

# F10 — Vahelduvvool. Reaalsed gaasid

| | |
|---|---|
| **Kursuse kood** | F10 (Flai2) |
| **Eelduskursused** | F3–F6 |
| **Õpetamise aeg** | 11. klass |
| **Hindamisviis** | Mitteeristav (arvestustöö) |

## Eesmärgid

Ohmi seadus vahelduvvooluringis. Elektromagnetlainete tekitamine. Elektrivool erinevates keskkondades. Reaalgaasi käsitlus.

## Põhiseosed

$$u = U_M\cos(\omega t + \varphi_0) \qquad i = I_M\cos(\omega t + \varphi_0) \qquad \omega = 2\pi f = \frac{2\pi}{T}$$

$$\cos\varphi = \frac{R}{Z} \qquad X_C = \frac{1}{\omega C} \qquad X_L = \omega L \qquad Z = \sqrt{R^2 + (X_L - X_C)^2}$$

$$I_M = \frac{U_M}{Z} \qquad I_{ef} = \frac{U_{ef}}{Z} \qquad N = UI = \frac{U_m I_m}{2} \qquad T = 2\pi\sqrt{LC}$$

$$\left(p + \frac{v^2 a}{V^2}\right)(V - vb) = vRT$$

## Õppesisu

Vahelduvvool. Resonants. Võnkering. Elektrolüüs. Gaaslahendus. Ülekandenähtused. Van der Waalsi võrrand.

## Praktilised tööd

1. Resonantsi uurimine võnkeringis
2. Mahtuvustakistus vahelduvvooluringis
3. Induktiivtakistus vahelduvvooluringis
4. Vase elektrokeemilise ekvivalendi määramine

# F11 — Pooljuhid. Fotomeetria. Megamaailm

| | |
|---|---|
| **Kursuse kood** | F11 (Flai3) |
| **Eelduskursused** | F7, F8 |
| **Õpetamise aeg** | 12. klass |
| **Hindamisviis** | Mitteeristav (arvestustöö) |

## Eesmärgid

Tuumareaktsioonide energeetiline käsitlus. Pooljuhtelektroonika ja fotomeetria alused. Doppleri efekt. Kepleri seadused. Erirelatiivsusteooria alused.

## Põhiseosed

$$E = mc^2 \qquad f = f_0\sqrt{\frac{1+v/c}{1-v/c}} \approx f_0(1+v/c)$$

$$v = c\frac{\Delta\lambda}{\lambda_0} \qquad T = 2\pi\sqrt{\frac{r^3}{GM}} \qquad E = \frac{I}{R^2}\cos\alpha$$

$$t = \frac{T}{\sqrt{1-\frac{v^2}{c^2}}} \qquad m_v = \frac{m_0}{\sqrt{1-\frac{v^2}{c^2}}} \qquad l_0 = \frac{l_v}{\sqrt{1-\frac{v^2}{c^2}}}$$

## Õppesisu

Tuumafüüsika. Pooljuhtelektroonika. Fotomeetria. Doppleri efekt. Taevamehaanika. Erirelatiivsusteooria.
