# F7 — Võnkumised ja lained. Geomeetriline- ja laineoptika

| | |
|---|---|
| **Kursuse kood** | F7 |
| **Eelduskursused** | F1–F3, F6 |
| **Õpetamise aeg** | 12. klass |

## Põhiseosed

$$x = X_m\cos\omega t \qquad \omega = \frac{2\pi}{T} = 2\pi f \qquad T = \frac{1}{f}$$

$$T = 2\pi\sqrt{\frac{l}{g}} \qquad T = 2\pi\sqrt{\frac{m}{k}} \qquad c = f\lambda$$

$$D = \frac{1}{f} \qquad \frac{1}{f} = \frac{1}{a} + \frac{1}{k} \qquad \Delta_{max} = 2k\frac{\lambda}{2}$$

$$\Delta_{min} = (2k+1)\frac{\lambda}{2} \qquad d\sin\alpha = k\lambda \qquad n = \frac{c}{v} \qquad \frac{\sin\alpha}{\sin\gamma} = \frac{n_2}{n_1} \qquad s = \frac{k}{a}$$

## Õppesisu

**Mehaaniline võnkumine:** Harmooniline võnkumine. Resonants. Vedrupendel. Matemaatiline pendel.

**Mehaanilised lained:** Lainepikkus. Heli. Doppleri efekt.

**Elektromagnetlained:** Elektromagnetlainete skaala. Raadioside. GPS.

**Laineoptika:** Interferents. Difraktsioon. Polarisatsioon. Dispersioon. Spektraalanalüüs.

**Geomeetriline optika:** Peegeldumine. Murdumine. Lääts. Läätse valem. Optilised süsteemid.

## Praktilised tööd

1. Raskuskiirenduse määramine matemaatilise pendli abil
2. Vedrupendli liikumise uurimine
3. Läätse optilise tugevuse määramine
4. Klaasi murdumisnäitaja määramine
5. Valguse lainepikkuse määramine difraktsioonvõre abil
6. Spektrite uurimine

# F8 — Kvantoptika. Aatomi- ja tuumafüüsika

| | |
|---|---|
| **Kursuse kood** | F8 |
| **Eelduskursused** | F6, F7 |
| **Õpetamise aeg** | 12. klass |

## Põhiseosed

$$E = hf \qquad hf = A + E_k \qquad E = mc^2 \qquad p = mc$$

$$\lambda = \frac{h}{mv} \qquad N = N_0 \cdot 2^{-\frac{t}{T}}$$

## Õppesisu

**Kvantoptika:** Plancki hüpotees. Fotoefekt. Einsteini valem. Footon. Tsooniteooria alused. Laser.

**Aatomifüüsika:** Rutherfordi katse. Bohri mudel. De Broglie hüpotees. Kvantmehaanika alused. Kvantarvud.

**Tuumafüüsika:** Aatomituum. Tuumajõud. Massidefekt. Seoseenergia. Radioaktiivsus. Poolestusaeg. Tuumareaktorid. Ioniseeriv kiirgus.

## Praktilised tööd

1. Tutvumine aatomimudelitega arvutisimulatsioonide abil
2. Radioaktiivse kiirguse neeldumise uurimine
3. Plancki konstandi määramine
4. Radioaktiivse lagunemise mudeldamine
